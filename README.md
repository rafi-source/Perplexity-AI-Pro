# Perplexica Lite: Advanced AI Research Assistant

Perplexica Lite is a sophisticated AI-powered search and research assistant designed for users who require deep, accurate, and up-to-date information. Powered by Google's most advanced models, it goes beyond simple search to provide a comprehensive, interactive research experience.

## Key Features

-   **Deep Research Mode:** A "Pro Mode" that conducts exhaustive, first-principles analysis for complex queries.
-   **Selectable Pro Models:** Choose from a range of Gemini models to tailor the AI's capabilities to your needs.
-   **Live Web Search:** Get the latest information from the web, with cited sources for every claim.
-   **Interactive Feedback:** Provide feedback on AI responses to help improve the model over time.
-   **Stop Generation:** Full control over the AI's responses with the ability to stop generation at any time.
-   **Code Support:** Code blocks with syntax highlighting and a "copy to clipboard" feature.
-   **File Attachments:** Attach images and other files for analysis.

## Technology Stack

-   **Frontend:** React, TypeScript, Vite
-   **AI:** Google Gemini API
-   **UI Components:** Lucide React

## Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up your environment variables:**
    -   Create a file named `.env.local` in the root of the project.
    -   Add your Gemini API key to the file:
        ```
        GEMINI_API_KEY=your_api_key_here
        ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:3000`.
